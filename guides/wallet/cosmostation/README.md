# Cosmostation

[Cosmostation](https://www.cosmostation.io/) 是用於 Cosmos 跨鏈、以太坊、ETH 第二層網絡解決方案及 EVM 生態系統的非託管多鏈加密貨幣瀏覽器擴展錢包。使用 Cosmostation 錢包你可以將自己的 LikeCoin 去中心化保存並自行掌握私鑰。

留意用戶暫時不能使用 Cosmostation 註冊 Liker ID。
