# NFT Portal

若作者不是使用 WordPress 網站，可到 **** [**NFT Portal**](https://app.like.co/nft/url) 網站貼上文章的網址。系統會自動抓取文章標題及把文章內容儲存到分散式檔案系統，並生成 NFT。

非 WordPress 用戶可以 iframe 的方法把 NFT Widget 嵌入文章，但需手動把 portal 生成的 ISCN 填作 widget 的參數。
