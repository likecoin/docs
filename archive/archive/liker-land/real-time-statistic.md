# 即時數據

{% hint style="danger" %}
2022/06/14 存檔。資訊已過期。
{% endhint %}

截至 2021年7月，[LikeCoin DAO](https://liker.land/getapp) 有 **1,700+ 名** [**讚賞公民**](../../../user-guide/civic-liker/)， 佔總數 **104,584+ Likers** 約 1**.6%**，共回饋了 **45mil+ LikeCoin** 給 **17,276+ 位創作者**於 **1,400+ 個媒體上**的 **519,568+ 筆內容**。‌
