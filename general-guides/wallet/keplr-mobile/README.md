# Keplr 手機錢包

Keplr Mobile 是 Keplr 的手機版錢包。通過移動設備裝置的便利，用戶可以委託通證、管理收益、參與治理及連接到 DeFi 應用程序。用戶亦可使用[ Keplr 瀏覽器錢包](../keplr/)管理你的 LikeCoin。
